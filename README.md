<h3>📝 Commit Convention</h3>
<div>
  <ul>
    <li>build : 빌드 파일 생성 및 수정</li>
    <li>chore : 개발 환경 세팅 및 패키지 설치</li>
    <li>design : 퍼블리싱 및 디자인 수정</li>
    <li>docs : 문서 수정</li>
    <li>feat : 새로운 기능 추가</li>
    <li>fix : 버그 수정</li>
    <li>refactor : 코드 리팩토링</li>
    <li>style : 코드 의미에 영향을 주지 않는 변경사항</li>
    <li>test : 테스트 코드 추가</li>
  </ul>
</div>
